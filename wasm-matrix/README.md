# 考えるべき仕様について

以下の流れで処理を行う
```mermaid
flowchart TD
    A[行列を作成する] --> B[2つの行列を関数に渡す] --> C{2つの行の長さ、列の長さが等しいか確認}
    --> |偽|D[エラーを出す]
    C --> |真| E[双方一行ずつ取り出す]　-->
    F[ルーチンの中で加算する] --> G{全行を取り出して計算したか} --> |偽| E
    G --> |真| H[結果を返す] 
```

# 実装
- 行列の型を定義する
```go
type Matrix [][]int 
// 整数型の配列の配列
```

- Goにはクラスの概念がないため、関数で定義する
    - 行列構造体の新規作成(interface使用)
    - 行列構造体の要素の取得(interface使用)
    - 行列の一行加算(interface不使用)
    - 行列の一行減算(interface不使用)
